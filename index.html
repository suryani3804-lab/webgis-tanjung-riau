WebGIS Fasilitas Umum dan Fasilitas Sosial

Sistem Informasi Geografis berbasis web untuk menyajikan informasi persebaran fasilitas umum dan fasilitas sosial di Kelurahan Tanjung Riau, Kota Batam.

Buka Peta Interaktif

Tentang WebGIS

WebGIS ini dikembangkan sebagai bagian dari kegiatan Project Based Learning (PBL) untuk menyajikan informasi geospasial mengenai fasilitas umum dan fasilitas sosial secara interaktif. Pengguna dapat melihat lokasi serta persebaran objek melalui peta berbasis web.
